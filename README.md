## 🚀 How to Run the Code Locally

### Prerequisites
Make sure you have the Java Development Kit (JDK) installed. You can check this by running:
```bash
java -version
```

### Compiling and Running
To compile and run any of the Java programs, open your terminal/command prompt and run the following commands:

1. **Compile the program:**
   ```bash
   javac filename.java
   ```
   *(e.g., `javac hello_World.java`)*

2. **Run the program:**
   ```bash
   java filename
   ```
   *(e.g., `java hello_World`)*
